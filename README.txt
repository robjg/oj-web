OJ Web 0.1.0
============

Web related components for Oddjob.

OJ Web provides:
- An HTTP Server to be used from within Oddjob.
- An HTTP client that can be used from within Oddjob.
- A Web Service that provides an API onto an Oddjob job tree.

Requirements
------------

Java 1.7

Installation
------------

To use with Oddjob either:

1. Unzip and include lib/*.jar in Oddjobs classpath

or:

2. Unzip into Oddjob's oddballs directory and restart Oddjob.
